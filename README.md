# Personal-Portfolio-Website
Portfolio Website

A personal portfolio website created using HTML and CSS to showcase skills, projects, and contact information in a clean and responsive layout.

Features
Responsive design for mobile and desktop
Navigation bar with smooth section links
Home, About, Skills, Projects, and Contact sections
Project showcase cards
Hover effects and modern UI design
Simple and easy-to-understand structure
Technologies Used
HTML5
CSS3
Project Structure
portfolio-website/
│
├── index.html
├── style.css
└── README.md
How to Run the Project
Download or clone the repository
Open the project folder in VS Code
Open index.html in any browser

You can also use the Live Server extension in VS Code for better viewing.

Purpose of the Project

The main purpose of this project is to practice frontend web development concepts and create a professional portfolio website for showcasing personal projects and technical skills.

What I Learned
Building webpage layouts using HTML
Styling webpages with CSS
Creating responsive designs using Flexbox and media queries
Adding hover effects and improving UI design
Organizing sections for better user experience
Future Improvements
